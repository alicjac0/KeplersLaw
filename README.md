# Overview

This project is an interactive simulation of planetary motion built using JavaScript and the HTML5 Canvas. It demonstrates the fundamental principles of orbital mechanics by modeling the motion of a celestial body under the influence of gravity.

The system computes motion through numerical integration, allowing orbital paths to emerge naturally from physical laws rather than being predefined. As a result, the simulation reproduces key characteristics of Keplerian motion, including elliptical orbits, non-uniform orbital velocity, and the relationship between orbital period and orbital radius.

### Controls
The Control panel allows user to:

* Adjust initial velocity
* Modify central mass
* Modify planet's placement
